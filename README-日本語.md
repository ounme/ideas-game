# BELIVEME: README

*パーソナリティ心理学を通じたアイデア伝播のエージェントベースシミュレーション*

## 概要

**BELIVEME** は、アイデアがどのように広がり、変容し、コミュニティを形成するかをモデル化します。エンジンは以下を組み合わせます：
- **パーソナリティ特性** (OCEAN + 知能)；  
- **アイデアのダイナミクス** (内容、感情的トーン、歪み)；  
- **社会的メカニズム** (影響力、適応、進化)。

## 主要コンポーネント

### 1. `Idea` クラス
伝播可能な概念を表し、以下を含みます：
- `core`：テキストの本質；  
- `effects`：パラメータへの影響（例：`{'C': +8, 'A': +5}`）；  
- `complexity`：0–100（伝達中の歪みに影響）；  
- `emotion`：感情的トーン（例：「畏敬」）；  
- `weight`：コミュニティ内でのアイデアの強さ（採用とともに成長）。

### 2. `Agent` クラス
各エージェントは 6 つの特性を持ちます（0–100）：
- **O** (Openness) — 新しい経験への受容性；  
- **C** (Conscientiousness) — ルール遵守、信頼性；  
- **E** (Extraversion) — 社会的活動；  
- **A** (Agreeableness) — 共感、親切さ；  
- **N** (Neuroticism) — 脅威への感受性；  
- **I** (Intelligence) — 批判的思考。

**コアメソッド**：
- `calculate_social_weight()`：影響力を計算（E×0.4 + C×0.3 + I×0.3）；  
- `transmit_idea()`：歪みと適応を伴ってアイデアを送信；  
- `adapt_idea()`：ソース/受信者の特性に基づいてアイデアを修正；  
- `receive_idea()`：フィルター付きで効果を適用（例：高 C は否定的変化に抵抗）；  
- `evolve_idea()`：伝達後、アイデアの複雑さと感情を更新。

### 3. `Community` クラス
エージェント集団とシミュレーションを管理：
- `add_idea()`：システムに新しいアイデアを注入；  
- `step()`：1 つのシミュレーションサイクルを実行（ソース → 受信者選択 → 伝達 → 更新）；  
- `update_history()`：平均特性値を記録；  
- `plot_history()`：パラメータの傾向を可視化。

## シミュレーションワークフロー

1. **初期化**  
   - `Community(size=N)` を作成；  
   - 初期 `Idea` インスタンスを追加。

2. **サイクル (`step()`)**  
   a. ランダムな**ソースエージェント**を選択（アイデアを持つ必要あり）；  
   b. ソースの信念からランダムな**アイデア**を選択；  
   c. 加重確率で**受信者**を選択（ソースの E × 受信者の A）；  
   d. **伝達**：  
      - 歪みを適用（アイデアの複雑さとソース I に基づく）；  
      - ソース/受信者の特性にアイデアを適応；  
      - 受信者が特性を更新しアイデアを採用；  
      - アイデアが進化（複雑さ/感情が変化）。  
   e. 特性平均値を記録。

3. **終了**  
   - `num_steps` 後、結果をプロットして統計を出力。

## 出力

1. **コンソールログ**  
   ```
   Шаг 0: средний O=49.4, C=48.9, E=50.0  
   ...  
   Симуляция завершена (100 шагов).
   ```

2. **グラフ**  
   - ステップごとの平均 OCEAN+I 値の折れ線グラフ。

3. **最終統計**  
   - エージェント全体での各特性の平均/最小/最大；  
   - アイデア採用率（例：*"Соблюдай завет"：22/30 エージェント*）；  
   - トップ 5 社会的リーダー（`social_weight` 順）。

## 使用方法

1. **依存関係をインストール**  
   ```bash
   pip install matplotlib
   ```

2. **シミュレーションを実行**  
   ```bash
   python main.py
   ```

3. **カスタマイズ**  
   - `__main__` ブロック内の `Community(size)` と初期アイデアを変更；  
   - アイデアの `effects`、`complexity`、`emotion` 値を調整。

## パラメータと調整

- **エージェント特性**：範囲内でランダムに初期化（例：O：30–70）；  
- **歪み**：`±0.1 × (complexity / 100)`、ソース I で減少、受信者 N で増加；  
- **適応ルール**：  
  - 高 A（ソース）→ コアテキストを柔らかくする；  
  - 低 O（受信者）→ 複雑さを下げる；  
  - 高 N（受信者）→ 感情を強化；  
- **アイデア進化**：伝達ごとに `weight × 1.01`；ソース O に基づく複雑さ ±5%。
