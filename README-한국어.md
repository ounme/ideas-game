# BELIVEME: README

*성격 심리학을 통한 아이디어 전파의 에이전트 기반 시뮬레이션*

## 개요

**BELIVEME**는 아이디어가 어떻게 확산되고, 변형되며, 커뮤니티를 형성하는지를 모델링합니다. 엔진은 다음을 결합합니다:
- **성격 특성** (OCEAN + 지능);  
- **아이디어 역학** (내용, 감정적 톤, 왜곡);  
- **사회적 메커니즘** (영향력, 적응, 진화).

## 주요 구성 요소

### 1. `Idea` 클래스
전파 가능한 개념을 나타내며 다음을 포함합니다:
- `core`: 텍스트 본질;  
- `effects`: 매개변수 영향 (예: `{'C': +8, 'A': +5}`);  
- `complexity`: 0–100 (전달 중 왜곡에 영향);  
- `emotion`: 감정적 톤 (예: "경외");  
- `weight`: 커뮤니티 내 아이디어의 강도 (채택과 함께 성장).

### 2. `Agent` 클래스
각 에이전트는 6개의 특성을 갖습니다 (0–100):
- **O** (Openness) — 새로운 경험에 대한 수용성;  
- **C** (Conscientiousness) — 규칙 준수, 신뢰성;  
- **E** (Extraversion) — 사회적 활동;  
- **A** (Agreeableness) — 공감, 친절;  
- **N** (Neuroticism) — 위협에 대한 민감성;  
- **I** (Intelligence) — 비판적 사고.

**핵심 메서드**:
- `calculate_social_weight()`: 영향력 계산 (E×0.4 + C×0.3 + I×0.3);  
- `transmit_idea()`: 왜곡 및 적응과 함께 아이디어 전송;  
- `adapt_idea()`: 소스/수신자 특성에 따라 아이디어 수정;  
- `receive_idea()`: 필터와 함께 효과 적용 (예: 높은 C는 부정적 변화에 저항);  
- `evolve_idea()`: 전달 후 아이디어 복잡성과 감정 업데이트.

### 3. `Community` 클래스
에이전트 집단과 시뮬레이션 관리:
- `add_idea()`: 시스템에 새로운 아이디어 주입;  
- `step()`: 하나의 시뮬레이션 사이클 실행 (소스 → 수신자 선택 → 전달 → 업데이트);  
- `update_history()`: 평균 특성 값 기록;  
- `plot_history()`: 매개변수 트렌드 시각화.

## 시뮬레이션 워크플로우

1. **초기화**  
   - `Community(size=N)` 생성;  
   - 초기 `Idea` 인스턴스 추가.

2. **사이클 (`step()`)**  
   a. 무작위 **소스 에이전트** 선택 (아이디어 보유 필요);  
   b. 소스의 신념에서 무작위 **아이디어** 선택;  
   c. 가중 확률로 **수신자** 선택 (소스의 E × 수신자의 A);  
   d. **전달**:  
      - 왜곡 적용 (아이디어 복잡성과 소스 I 기반);  
      - 소스/수신자 특성에 아이디어 적응;  
      - 수신자가 특성 업데이트하고 아이디어 채택;  
      - 아이디어 진화 (복잡성/감정 변화).  
   e. 특성 평균값 기록.

3. **종료**  
   - `num_steps` 후, 결과 플롯하고 통계 출력.

## 출력

1. **콘솔 로그**  
   ```
   Шаг 0: средний O=49.4, C=48.9, E=50.0  
   ...  
   Симуляция завершена (100 шагов).
   ```

2. **그래프**  
   - 단계별 평균 OCEAN+I 값의 라인 차트.

3. **최종 통계**  
   - 에이전트 전체의 각 특성에 대한 평균/최소/최대;  
   - 아이디어 채택률 (예: *"Соблюдай завет": 22/30 에이전트*);  
   - 상위 5명의 사회적 리더 (`social_weight` 순).

## 사용법

1. **의존성 설치**  
   ```bash
   pip install matplotlib
   ```

2. **시뮬레이션 실행**  
   ```bash
   python main.py
   ```

3. **사용자 지정**  
   - `__main__` 블록에서 `Community(size)` 및 초기 아이디어 수정;  
   - 아이디어의 `effects`, `complexity`, `emotion` 값 조정.

## 매개변수 및 조정

- **에이전트 특성**: 범위 내에서 무작위로 초기화 (예: O: 30–70);  
- **왜곡**: `±0.1 × (complexity / 100)`, 소스 I로 감소, 수신자 N으로 증가;  
- **적응 규칙**:  
  - 높은 A (소스) → 핵심 텍스트 부드럽게 함;  
  - 낮은 O (수신자) → 복잡성 낮춤;  
  - 높은 N (수신자) → 감정 강화;  
- **아이디어 진화**: 전달당 `weight × 1.01`; 소스 O 기반 복잡성 ±5%.
