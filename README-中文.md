# BELIVEME: README

*基于代理的思想传播模拟，通过人格心理学*

## 概述

**BELIVEME** 建模思想如何传播、转化和塑造社区。引擎结合：
- **人格特质** (OCEAN + 智力)；  
- **思想动态** (内容、情感基调、扭曲)；  
- **社会机制** (影响力、适应、演化)。

## 主要组件

### 1. `Idea` 类
表示可传播的概念，包含：
- `core`：文本本质；  
- `effects`：参数影响（例如，`{'C': +8, 'A': +5}`）；  
- `complexity`：0–100（影响传播过程中的扭曲）；  
- `emotion`：情感基调（例如，"敬畏"）；  
- `weight`：思想在社区中的强度（随采纳而增长）。

### 2. `Agent` 类
每个代理有 6 个特质（0–100）：
- **O** (Openness) — 对新体验的接受度；  
- **C** (Conscientiousness) — 遵守规则、可靠性；  
- **E** (Extraversion) — 社交活动；  
- **A** (Agreeableness) — 同理心、友善；  
- **N** (Neuroticism) — 对威胁的敏感性；  
- **I** (Intelligence) — 批判性思维。

**核心方法**：
- `calculate_social_weight()`：计算影响力（E×0.4 + C×0.3 + I×0.3）；  
- `transmit_idea()`：发送带有扭曲和适应的思想；  
- `adapt_idea()`：根据源/接收者特质修改思想；  
- `receive_idea()`：应用带有过滤器的效果（例如，高 C 抵制负面变化）；  
- `evolve_idea()`：传播后更新思想复杂性和情感。

### 3. `Community` 类
管理代理群体和模拟：
- `add_idea()`：将新思想注入系统；  
- `step()`：运行一个模拟周期（源选择 → 接收者选择 → 传播 → 更新）；  
- `update_history()`：记录平均特质值；  
- `plot_history()`：可视化参数趋势。

## 模拟工作流程

1. **初始化**  
   - 创建 `Community(size=N)`；  
   - 添加初始 `Idea` 实例。

2. **周期 (`step()`)**  
   a. 选择随机**源代理**（必须持有思想）；  
   b. 从源的信念中选择随机**思想**；  
   c. 通过加权概率选择**接收者**（源的 E × 接收者的 A）；  
   d. **传播**：  
      - 应用扭曲（基于思想复杂性和源 I）；  
      - 思想适应源/接收者特质；  
      - 接收者更新特质并采纳思想；  
      - 思想演化（复杂性/情感改变）。  
   e. 记录特质平均值。

3. **终止**  
   - `num_steps` 后，绘制结果并打印统计数据。

## 输出

1. **控制台日志**  
   ```
   Шаг 0: средний O=49.4, C=48.9, E=50.0  
   ...  
   Симуляция завершена (100 шагов).
   ```

2. **图表**  
   - 平均 OCEAN+I 值随步骤变化的折线图。

3. **最终统计**  
   - 每个特质在代理中的平均值/最小值/最大值；  
   - 思想采纳率（例如，*"Соблюдай завет"：22/30 代理*）；  
   - 前 5 名社会领袖（按 `social_weight`）。

## 使用

1. **安装依赖**  
   ```bash
   pip install matplotlib
   ```

2. **运行模拟**  
   ```bash
   python main.py
   ```

3. **自定义**  
   - 修改 `__main__` 块中的 `Community(size)` 和初始思想；  
   - 调整思想的 `effects`、`complexity` 和 `emotion` 值。

## 参数与调整

- **代理特质**：在范围内随机初始化（例如，O：30–70）；  
- **扭曲**：`±0.1 × (complexity / 100)`，由源 I 减少，由接收者 N 增加；  
- **适应规则**：  
  - 高 A（源）→ 软化核心文本；  
  - 低 O（接收者）→ 降低复杂性；  
  - 高 N（接收者）→ 强化情感；  
- **思想演化**：每次传播 `weight × 1.01`；基于源 O 的复杂性 ±5%。
